# PKF_BK_Tasks
 В этом репозитории находится 4 программы  
 1) Калькулятор, реализованный с помощью технологии WPF с применением паттерна MVVM. (Ветка Calculator)
 2) Мессенджер между двумя и более приложениями, через который можно отправлять сообщения и файлы. 
 Это клиент-серверное приложение, реализованное на классе Socket. Сервер - консольное приложение, 
 а клиент - WPF приложение с применением паттерна MVVM. (Ветка TCP_Messenger)
 3) Интерфейс на Unity, через который можно взаимодейстовать с объектами на сцене. (Ветка Interface)
 4) Мини игра куб-лабиринт на Unity. (Ветка CubeMaze)
